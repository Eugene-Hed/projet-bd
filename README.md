# Système de Gestion des Établissements

Un système complet de gestion des établissements permettant de centraliser et administrer les informations relatives aux établissements.

## Fonctionnalités

- Gestion des établissements
- Gestion des utilisateurs
- Administration des données
- Interface web intuitive

- Base de données SQL
- HTML/CSS pour l'interface web
- JavaScript pour l'interaction

1. Cloner le repository
2. Configurer l'environnement
3. Importer le schéma de la base de données
4. Lancer l'application
