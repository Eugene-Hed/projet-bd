# Module de Gestion du Personnel

## Tables Principales

### 1. utilisateurs
Table de base pour tous les membres du personnel.

### 2. enseignants
Informations spécifiques aux enseignants.

### 3. personnel_administratif
Informations pour le personnel administratif.

### 4. contrats_travail
Gestion des contrats de travail.

### 5. paiements_salaires
Gestion des paiements de salaires.
